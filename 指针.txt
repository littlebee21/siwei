指针
    运算符号
        解引用操作符/间接访问运算符*
        引用&
        +- 运算指针指向的类型的大小：连续空间才有意义，类型相同才有意义
        强制类型转换
        取差值：
    定义
        int pa;定义一个变量pa, 
        int *p;定义一个指针变量p
        
        1,int *p1[3];   // 指针数组
            中括号[]的优先级高，因此与p1先结合，表示一个数组，
            这个数组中有3个元素，这3个元素都是指针，
            它们指向的是int型数据。
        2,int (*p2)[3]; // 数组指针
            小括号让p2与*结合，表示p2是一个指针，
            这个指针指向了一个数组，
            数组中有3个元素，
            每一个元素的类型是int型。
        
    赋值
        int num = 7;    int *p = &num 
    初始化
        注意:指针变量的初始化和赋值的方式是不同的:
            初始化: int*p = &a;
            赋值: p = &a;

    
    复合表示内容原理
        数组指针：指针数组：
        函数指针，指针函数：
        按照优先级进行结合
            ()最高
            []其次
            *其次
            int其次
        结合的意义
            1,先定性:先结合确定自己所属的类型
            2,在定属性:再结合确定自己作为变量的 存储类型 + 指向内容

        三大地址特例
            1,字符串字面量作为右值时，就代表这个字符串在内存中的首地址；
            2,数组名就代表这个数组的地址，也等于这个数组的第一个元素的地址;
            3,函数名就代表这个函数的地址。


    const指针
        指针指向的类型    
        void 指针
            void类型的指针可以赋值给其他类型的指针
            其他类型的指针也可以复制给void类型的指针
            不同类型的指针不能互相赋值
            优点
                通用性强
            缺点
                在指针运算的时候需要强制转化为有数据类型的指针
    野指针
            规范：
                在指针初始化的时候要进行赋值，赋值成空指针
                而不是在赋值的时候进行第一次赋值
                segmentation fault 段错误，非法访问内存
    空指针


    
    如何在编程中是思考指针：
            1，关注指针所指向的内存空间的内容（*p），不要关注指针p本身的内容
    如何使用指针：
            1，分开定义，让每个指针都有自己的名字
            2，使用i，j等整形变量+指针，来进行地址的取值
            3，可以定义数组，然后使用指针的方式使用数组；
           错误:
            1,不能在局部变量里定义指针作为缓存； 

    如何使用多维指针
            1，明确指针的数据结构  p[]
                只要是能够指代其他内容的，都是指针
            2，明确指针的指向类型 *p[]
            3，明确指针的指向类型
