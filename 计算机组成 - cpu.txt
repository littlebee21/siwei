        前端总线
            连接 CPU 与内存，负责数据传输
            CPU 中的内存控制器管理 CPU 与内存的数据交换
            前端总线带宽 = 内存控制器的工作频率 x 内存位宽 ( 双通道内存位宽为内存本身宽度 x 2 ) = 通常为 1600 MHz x 64 bit = 12.8 GB/s，由于内存控制器可以搭配低频率内存工作，所以实际 FSB 带宽由实际内存工作频率决定
        内部缓存
            工作频率与 CPU 工作频率相同，容量很小
            划分为 L1/L2/L3 级缓存，容量依次增大，频率依次降低
