        vim插入注释

            进入v模式，选中然后按大写I，然后 <c-c> control+C

            注释到文件头的位置
            注释到文件尾的位置
        vim批量缩进
            ： 2,9>
        字符编辑
            x 删除光标所在处的字符
            #x 删除#个字符
            xp 将光标所在的字符和后面的字符互换位置
            ~ 将光标所在的位置大小写互换
            J 删除当前行的换行符
        替换
            r 替换当前光标所在处的字符
            R 进入替换模式
           D 相当于d$
        复制
            常用的复制命令
                y 也可以跟字符跳转结合
                yy    复制游标所在行整行 
                2yy或y2y    复制 2 行
                y^    复制至行首，或y0
                y$    复制至行尾
                yw    复制一个word
                y2w    复制两个word 
                yG    复制至文件尾
                y1G    复制至文件首
            光标移动下的复制
                要选中内容进行复制，先在命令模式下按 v 进入 Visual Mode，然后用方向键 或 hjkl 选择文本，再按 y 进行复制。
        移动
            zt 将光标所在的行移动到屏幕的顶端
            zb 将光标所在的行移动到屏幕的底端
            zz 将光标所在的行移动到屏幕的中间
        删除
            d 删除，需要跟字符跳转结合
            d$ 删除到行尾,删除当前位置
            d0 删除到行首,当前位置不删除
            d^ 删除到为空字符
            dw 删除一个单词
            de 向后删除一个单词
            db 向前删除一个单词
            dd 删除整行
            #dd 直接删除#行
            dG 删除到结尾
            djj 删除到开头
            
        粘贴
            p 如果复制的是整行的话,则粘贴到当前光标所在行的下一行,否则粘贴光标后面如果整行的话，则粘贴到当前光标的下一行，否则粘贴到光标的前面
        改变,删除以后并将模式切换成插入
            c 可以跟光标的跳转做结合
            cc 删除整行并插入
            C 相当于c$
        #ialex esc将alex 粘贴#次

find 会从 path 中搜索文件。所以在使用find之前一定要配置一下path变量（不是PATH环境变量）。
我们在 blog/ 目录下打开vim, 执行：
set path=.,page/,src/
然后当我们想要打开main.java时，只需要输入：

find main.java 或
find m[TAB]
