git工作流-同步主线




 1）切换到master
1
 git checkout master
1
2）从remote master拉取最新代码到local master
1
  git pull
1
3）切换到dev
1
   git checkout dev
1
4）rebase master 到 dev（将主支代码合并到分支）
1
   git rebase master
1
5.如果有代码冲突，则解决。
