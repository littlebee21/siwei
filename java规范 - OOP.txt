    (四) OOP规约	9
        权限
            final关键字
                类：不可继承
                对象：不可修改引用：POJO的域变量
                方法：不可以被修改：POJO的setting方法
            成员变量
                类方法变量尽量控制使用范围
                3） 类非 static 成员变量并且与子类共享，必须是 protected。
                4） 类非 static 成员变量并且仅在本类使用，必须是 private。 
                5） 类 static 成员变量如果仅在本类使用，必须是 private。
            常量
                常量声明为**static final**，并以大写命名
            构造方法
                静态类、单例类、工厂类将它们的**构造函数置为private**
                1） 如果不允许外部直接通过 new 来创建对象，那么构造方法必须是 private。 
                2） 工具类不允许有 public 或 default 构造方法。 
        类
            static静态类的使用
                1. 静态类 不允许实例 化，为静态类添加一个私有构造函数。
                2. 静态类必须满足：无状态、不可变、不允许继承、线程安全。
                3. 确保使用静态类不会给程序带来副作用。通常静态类会作为工具类提供，例如过滤列表等。
            extend继承的使用
                优先考虑组合而不是继承。
            interface的使用
                接口中不要有常量
        修饰符
            标记
                复写方法Override
                接口过时加deprecated不再使用
                不修改方法而是重写方法
