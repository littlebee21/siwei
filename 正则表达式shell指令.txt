正则表达式shell指令
        支持正则表达式的工具软件
            grep
                选项
                    -n
                        加上行号
                    --color
                        显示颜色
                    -An
                        后 n 行
                    -Bn
                        前 n 行
                    -v
                        不包含模式字符串
                    -i
                        忽略大小写
                    -E
                        使用扩展的正则表达式
                模式字符串 
                    [^b]
                        反向选择b
                            ^出现在中括号中
                    a-z
                        从a开始，到z 结束的所有字符
                    [:lower:]
                        相当于
                            a-z
                    [[:lower:]]
                        相当于
                            [a-z]
                    "^[a-z]"
                        ^表示字符串开头
                            ^出现在中括号之外，模式字符串开头
                    "[a-z]$"
                        $表示字符串结尾
                    .
                        任意一个字符
                    重复
                        *
                            重复0到任意多次
                        \{n, m\}
                            前面的字符重复n, m 次
                            { 和 } 字符在 shell中有特殊含义，因此需要使用反斜杠进行转移
                            子主题
                        \{n\}
                            重复 n 次
                        \{n, \}
                            至少重复n次
            sed
                作用
                    按行进行文本的删除、添加和替换
                syntax
                    $sed 选项 动作
                        选项
                            -n
                                安静模式
                            -e
                                在命令行界面进行sed动作的编辑, -e 可以省略，效果一样
                            -f
                                把sed动作放在该选项后面的文件中
                            -r
                                使用扩展的正则表达式
                            -i
                                直接修改读取的文件内容，不在屏幕上输出
                        动作
                            syntax
                                n1, n2 function
                                    n1,n2
                                        要处理的行号范围
                                        行号为$ 表示最后一行
                                    function
                                        a 字符串
                                            字符串作为一行新增
                                                位置为当前行的下一行
                                        i 字符串
                                            插入一行字符串
                                                位置为当前行的上一行
                                        增加多行用反斜线转义换行符
                                        c 字符串
                                            用字符串取代n1, n2之间的行
                                        d
                                            删除 n1,n2 行
                                        p
                                            打印某个选择的数据，通常与 -n选项一起执行
                                            例
                                                $ nl /etc/passwd | sed -n '5,7p' 
                                                    打印第5,7行
                                        s正则表达式
                                            使用正则表达式进行替换
                                            syntax
                                                sed 's/要被取代的字符串/新的字符串/g'
                                                    新的字符串为空则不需要在这个位置上写任何内容
            awk
                处理并输出一行中的多个字段
                syntax
                    $awk '条件类型1{动作1} 条件类型2{动作2} ...' filename
                        内置变量
                            NF
                                一行中的字段总数
                            NR
                                正在处理的当前行是第几行
                            FS
                                目前的分隔字符
                                    默认是空白符
                        条件类型
                            逻辑运算符
                                >
                                <
                                >=
                                <=
                                ==
                                !=
                        {}中的可以有多个命令作为动作，动作之间需要使用 ";" 分隔或用 Enter 键隔开各个动作
                        {} 中也可以支持使用 if 作为条件，格式为 { if (条件表达式) 动作...}
                    例
                        $ last -n 5 | awk '{print $1 \t $3}'
                            打印第1个字段和第3个字段，以tab键分隔
                            n=1,2,3...
                                $n表示第n个字段
                            n=0
                                $0 表示一行数据
                        $ awk 'NR>=2{ total=$1+$2+$3 printf  "%10d%10d%10d%10.2f\n", $1,$2,$3,total}'
                            对第2行及后面的行进行处理
                            total 是 awk 中的变量，不需要加$符号
                    关键字
                        用途
                            awk 以行为单位进行扫描
                        BEGIN
                            在行扫描开始之前执行的动作
                            例
                                'BEGIN {FS=":"}'
                                    在开始行扫描之前就把FS设置为 ":"
                                    可以避免在第一行使用默认的空格作为FS
                        END
                            在行扫描结束之后执行的动作
        影响正则表达式匹配结果的因素
