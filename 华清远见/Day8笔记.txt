	[4] 基础知识 
		(1) 进位计数制
			1) 基本概念:凡是以进位进行计数的数制称为进位计数制，简称进制
			2) 二进制、八进制、十六进制
		(2) 数制转换
			1) 二进制、八进制、十六进制 ... 转换成 十进制  
				按权相加
					整数部分: 从右向左,每位的位权 N^(i-1)
					小数部分: 从左向右,每位的位权 N^(-j)
			2) 十进制 转换 二进制、八进制、十六进制 ...
				整数部分: 除N求余,逆序排序 
				小数部分: 乘N取整,顺序排序
			3) 二进制 与 八进制 转换  
				从低位到高位,每3位二进制数可以转换成1位八进制数,高位不足补0.
				从低位到高位,每1位八进制数可以转换成3位二进制数
			4) 二进制 与 十六进制 转换  
				从低位到高位,每4位二进制数可以转换成1位十六进制数,高位不足补0.
				从低位到高位,每1位十六进制数可以转换成4位二进制数	
		(3) 整数的编码方式 
			1) 原码 
				最高位为符号位 : 0 表示是正数, 1 表示为负数
				剩余为为数值位 : 存该数的绝对值
			2) 反码 
				最高位为符号位 : 0 表示是正数, 1 表示为负数
				对于正数 反码等于原码 
				对于负数 符号位不变，其余位全部取反
			3) 补码
				最高位为符号位 : 0 表示是正数, 1 表示为负数
				对于正数 补码等于原码 
				对于负数 符号位不变，其余位全部取反, 再加1
			4) 求补运算 
				所有位全部取反，再加1
			5) 计算机使用的是补码存储有符号整数

	[6] 变量 
		(1) 什么是变量
			所谓变量是指在程序运行过程中其值可以改变的量。使用变量时必须“先定义，后使用”。
		(2) 定义变量  
			数据类型 变量名;
			eg:  
				int num;
				float 单精度浮点型
				char 字符型
		(3) 变量的实质  
			内存中的一段存储空间，变量名就是这个空间的名字，用户可以通过变量名访问该空间。
		(4) 变量的赋值与初始化  
			运算符 "="
			1) 定义的同时初始化  
				int num = 15;
			2) 先定义变量，后初始化  
				int num; 
				num = 56;
		(5) 变量的地址  
			地址(内存): 计算机内存中每个字节的空间都一个编号，这个编号称为地址
			变量的地址：变量在内存中总占用几个连续的字节，开始字节的地址，就是变量的地址。
			
			运算符 "&"  取地址符
			&变量名;

	[7]	标准输入输出
		(1) 输出 printf 
			#include <stdio.h>							//使用prinf函数需要包含stdio.h			

			int printf(const char *format, ...);
			函数参数: 
				format : 格式控制字符串(用于指定输出数据格式)，包含两种信息
					a.普通字符：原样输出
					b.格式说明符：%[修饰符]格式字符，用于指定输出格式(格式说明符会被替换成 不定参数所指定的值 )
								%d 		表示该位置的数据是个整型数据
								%f		表示该位置的数据是个浮点型数据
								%c		表示该位置的数据是个字符型数据
					  ... : 不定参数，要输出的数据（可以没有，多个时以“,”分隔, 与格式说明符一一对应）
				示例:  
					int a, b;
					a = 15; 
					b = 20;
					printf ("a = %d, b = %d\n", a, b);
			
		(2) 输入 scanf 
			#include <stdio.h>

			int scanf(const char *format, ...);
			函数参数: 
				format : 格式控制串(用于指定输入数据格式),包含两种信息
					a.普通字符 ：原样输入 
					b.格式说明符：%[修饰符]格式字符，用于指定输入数据格式(输入的数据将会被保存到不定参数所指定的变量中)
						%d 		表示该位置的数据是个整型数据
						%f
						%c
				… : 不定参数，输入变量的地址表（多个时以“,”分隔）		
				
				示例:  
					int a, b;		
					
					scanf ("a=%d,b=%d", &a, &b);
					scanf ("%d,%d", &a, &b);
				注意: 格式说明符之间不指定分隔符，默认分隔符 "space", "Enter", "Tab"
					scanf ("%d%d", &a, &b);

2. 常量
	[1] 什么是常量
		在程序执行过程中，其值不被改变的量。常量主要分为两类：
		(1) 直接常量：直接引用的数字等；
			字符常量	'A' , 'a'
			整型常量	10 100 0b1101  0664 0xaf
			实型常量	3.14 5.69
			字符串常量 	"hello world" "100"
		(2)符号常量：使用标识符来代替一个数值（常见的：宏定义常量 和 常变量）
		
	[2] 宏定义
		#define 也是C语言预处理命令的一种，被称为宏定义指令。
		所谓宏定义，就是用一个标识符来表示一个文本，如果在后面的代码中出现了该标识符，那么就全部替换成指定的文本。
			
		宏定义基本结构: 
			#define 	宏		替换体
				|		|			|
				|		|		一旦预处理器在程序中找到宏的实例后，就会用替换体代替该宏
				|	一个标识符(一般都会大写)，在代码中会被替换成指定的值
			宏定义指令 
		
		从宏变成最终替换文本的过程称为宏展开(macro expansion)。
		注意：宏定结束后不要添加分号;
		(1) 类对象宏(object-like macro 不带参宏) 
			#define 宏名 宏值
			例如：
				#define MAX 10
				#define PI 3.1415926
		(2) 类函数宏(function-like macro 带参数宏)
			#define 宏名(参数列表) 宏值
			例如：
			#define S(r) PI*r*r
			示例: 使用宏定义PI来定义3.1415926，定义宏定义，计算圆的面积			
				#include <stdio.h>

				#define PI 3.1415926
				#define S(r) PI * r * r 

				int main(int argc, const char *argv[])
				{
					int r;
					float s;

					printf("please input r :\n");
					scanf("%d", &r);

					s = S(r);

					printf("s = %f\n", s);
					
					
					return 0;
				}

		练习:租船问题。
			写程序计算游客需要付的租船的费用。租船每小时30元，押金100元。
			游客输入租船时间，计算出租船费用。租船费用=时间*每小时钱数+押金。
			要求押金与每小时钱数使用宏定义，租船费用公式用宏定义表示。
			
			#include <stdio.h>

			#define YAJIN 100
			#define PERHOUR 30
			#define MONEY(t) YAJIN+PERHOUR*t

			int main(int argc, const char *argv[])
			{
				int hour;
				int money;

				printf("please input hour :\n");
				scanf("%d", &hour);

				money = MONEY(hour);

				printf("Money : %d\n", money);
				
				return 0;
			}

		注意:　宏定义仅仅只是简单的替换，使用足够多的圆括号来确保运算和结合的正确顺序
			
	[3] 常变量 
		常变量(只读变量)：变量值不可改变的量，使用const修饰
		定义变量:
			const 数据类型 变量名 = 初值;
		示例: 
			const int a = 10;	//定义一个常变量
			a = 9;				//报错 ❌
		注意：
			const修饰后的变量会变成只读，因此无法再次赋值。因此初始化常变量需要在定义时直接赋值。

3. 数据类型 
	[1] 基本数据类型：C语言中已经定义的一种类型，基本数据类型最主要的特点是，其值不可以再分解为其它类型。
		整型 int
		字符型 char
		浮点型：
			单精度浮点型 float
			双精度浮点型 double
	[2] 构造数据类型：是根据已定义的一个或多个数据类型用构造的方法来定义的。也就是说，一个构造类型的值可以分解成若干个“成员”或“元素”。
		数组 
		结构体 struct 
		联合体/共用体 union 
		枚举型 enum 
	[3] 指针类型 : 指针是一种特殊的，同时又是具有重要作用的数据类型。其值用来表示某个量在内存储器中的地址。
	[4] 空类型: 表示什么都不是，常用于函数返回值或者定义空指针。
			void
